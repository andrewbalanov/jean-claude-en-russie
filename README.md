# 🍷🐻 Jean-Claude en Russie

Мини-игра для изучения **русского языка** — сделана для друга-француза, который производит и продаёт вино в России.
Un petit jeu pour apprendre le **russe**, фан, с юмором и винишком 🇫🇷❤️🇷🇺

## 🎮 Играть / Jouer
👉 **https://andrewbalanov.github.io/jean-claude-en-russie/**

Открывается прямо в браузере телефона или компьютера. Ставить ничего не надо, работает офлайн, прогресс сохраняется на устройстве.

## ✨ Что внутри
- **Заставка в стиле Dendy/NES** — Жан-Клод в берете с багетом у Кремля.
- **Платформер «à la Mario»** — беги, прыгай на медведей 🐻, собирай бутылки 🍷 (каждая = русское слово), доберись до Кремля 🏰. В конце — мини-квиз по собранным словам.
- **Уроки-карточки** — с транскрипцией, переводом и подсказками.
- **Квиз** — тренировка RU ↔ FR с «жизнями».
- **Озвучка** произношения + разговорник (шпаргалка).
- **Геймификация** — XP, 7 уровней (от «Турист с чемоданом вина» до «Французский медведь»), бейджи, ежедневная цель и стрик.

## 📚 Темы
🤝 Бизнес и продажи вина · 🍷 Дегустация · 😎 Сленг и юмор · 🧭 Выживание в России

## 🛠️ Как это устроено
Один самодостаточный файл `index.html` (HTML + CSS + JS, без зависимостей). Canvas для платформера, Web Speech API для озвучки, localStorage для прогресса.

## 🚀 Публикация (GitHub Pages)
Settings → Pages → Source: *Deploy from a branch* → Branch: `main`, папка `/ (root)` → Save.
